# pangud-dashboard

## 安装依赖

```bash
npm install
```

### 开发环境编译与热加载

``` bash
npm run serve
```

### 线上环境编译

``` bash
npm run build
```

### 运行单元测试

``` bash
npm run test:unit
```

### 运行端到端测试

``` bash
npm run test:e2e
```

### 代码检测与修复

``` shell
npm run lint
```
